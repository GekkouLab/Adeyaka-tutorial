# 使用命令

命令是在游戏中访问功能提供的接口。

插件主体的命令以 /adeyaka 开头，云相关的以 /adeyaka\_cloud 开头

分别可简略为 /ade 和 /adec

下面对每条子命令作介绍。

### /adeyaka info

功能：展示服务器当前安装的 Adeyaka 的信息

示例：/ms info

输出：

```
Adeyaka >>
>> 正在展示信息:
>> 作者: 紅葉 (KouyouX)
>> 版本: 1.0.0
```

### /adeyaka reload

别名：/ms r

功能：重载所有配置文件和脚本文件

示例：/ms r

输出：

```
Adeyaka >>
>> 正在重载
>> 已重载 配置文件
>> 已重载 脚本文件
>> 重载完毕
```
